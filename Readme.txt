
# Bibliothèque

Ce projet C# de gestion de bibliothèque permet d'enregistrer des abonnés, des livre, d'effectuer des emprunts et des retours de livres,et faire des verification grace a des requete.

## Fonctionnalités

- Enregistrement des abonnés : Les informations telles que le nom, le prénom, la date de naissance et le numéro d'abonné sont enregistrées pour chaque abonné.
- Enregistrement des livres : Les informations telles que le titre, l'auteur, le nombre de pages et la date de publication sont enregistrées pour chaque livre.
- Emprunt de livres : Les abonnés peuvent emprunter des livres sans limite 
- Retour de livres : Les abonnés peuvent retourner les livres empruntés.
- requete : Il est possible de vérifier la disponibilité d'un livre les historique d'emprunt d'un abonne
- enregistrement permanent automatique dans un fichier json 

## Configuration requise

- Microsoft .NET Framework (version 6.0)
- Visual Studio (version X.X.X) ou tout autre environnement de développement C#

## Installation et exécution

1. Clonez ou téléchargez le projet depuis le dépôt GitHub : [lien_vers_le_projet](lien_vers_le_projet)
2. Ouvrez le projet dans votre environnement de développement C# (par exemple, Visual Studio).
3. Compilez le projet pour vérifier qu'il n'y a pas d'erreurs de compilation.
4. Exécutez l'application.

## Utilisation

1. Lancez l'application.
2. Utilisez les différentes fonctionnalités pour enregistrer des abonnés, enregistrer des livres, effectuer des emprunts et des retours, et vérification grace au requete.
3. Fermez l'application lorsque vous avez terminé (Enregistrement automatique).
